# 解决 Fcitx 输入法在一些 qt5 程序下无法使用的问题。
我提供的 `libfcitxplatforminputcontextplugin.so` 都是在 Debian sid 中编译通过。详细支持的应用和 QT 版本见文件夹下的 `README` 内容。
目前支持应用：

- Mendeley Desktop
- RStudio
- Mathematica 12
